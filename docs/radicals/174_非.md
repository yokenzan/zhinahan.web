# 非部

## 非

- 音飛(Fei)平聲
    - 不是也。事物之有定義者。不合則謂之非。
    - 過失也。如有過失而回護掩飾者曰文過飾非。
    - 詆毀也。如言非聖無法。
    - 反設之辭。猶言未嘗。（孟子）城非不高也。
    - 洲名。阿非利加之簡稱。

## 啡

- 本音普海切。唾聲。今借為飲料之咖啡、及藥品之嗎啡字。並讀若非。

## 靠

- 音犒(Kao)去聲
    - 依他物以為安固之義。俗亦謂依附曰依靠。

## 靡

- 音米(Mi)上聲
    - 偃也。因勢之所趨相從而盡倒也。如言從風而靡。風靡一時。
    - 侈也。如奢靡、浮靡之類。
    - 無也。（詩）靡室靡家。
    - 靡靡、猶遲遲。（詩）「行邁靡靡」。又風俗敗壞也。（書）商俗靡靡。
