# 用部

## 用

- 讀如容(Yung)去聲
    - 體之對待字。事物之本能謂之體。其見諸施行、發為事業、謂之用。如作用、功用。
    - 任使也。如言用人行政。
    - 財用也。如謂國之財政曰國用。
    - 器用也。如器物亦曰用具。
    - 資以動作謂之用。如用心、用力、運用、動用。
    - 助詞。以也。是以亦或作是用。又爰也。於是之意。（左傳）「我敝邑用不敢保聚」。猶言於是不敢保聚也。
    - ［解字］本作用。左上缺不連。甫甬等字皆同。

## 甪

- 音鹿(Lu)入聲
    - 甪里先生、漢時四皓之一。
    - 甪里。鎮名。在江蘇吳縣。

## 甫

- 音斧(Fu)上聲
    - 男子之美稱也。如孔子為尼甫。周大夫有嘉甫。宋大夫有孔甫。古本作父。
    - 表字曰甫。故問人之字曰台甫。
    - 稱父曰甫。如謂人之父曰尊甫。
    - 方始也。如言甫有端倪之類。

## 甬

- 音勇(Yung)上聲
    - 甬道。尊貴者出行之路。築其地高出於兩邊者。
    - 江名。在浙江鄞縣。故簡稱鄞縣為甬。

## 甯

- 音佞(Ning)去聲
    - 所願也。如俗言甯可如此。

- 泥形切平
    - 與寧同。安也。
