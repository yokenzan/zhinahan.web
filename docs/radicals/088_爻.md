# 爻部

## 爻

- 音肴(Hsiao)平聲
    - 卦爻也。易卦有六爻。爻。交也。

## 爽

- 讀如霜(Shuang)上聲
    - 明也。天將明未明之時。謂之昧爽。
    - 豁達不拘滯也。如言開爽之類。
    - 清朗也。如言神清氣爽。
    - 無所留滯之意。如言爽快、爽利。
    - 失誤也。如失約謂之爽約。
    - 爽然。失意貌。如言爽然若失。

## 爾

- 讀如耳(Er)上聲
    - 汝也。（論語）惟我與爾有是夫。
    - 語助也。詞之終也。（孟子）然而無有乎爾。則亦無有乎爾。
    - 爾爾。猶言如此也。如言聊復爾爾。
