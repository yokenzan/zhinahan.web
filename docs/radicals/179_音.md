# 音部

## 音

- 音陰(Yin)平聲
    - 聲發而有清濁高下之節也。聲成文謂之音。見（禮記）。

## 韶

- 音召(Shao)平聲
    - 虞舜之樂也。
    - 美也。如韶華、韶光。皆言優美之景光也。

## 韻

- 音運(Yun)去聲
    - 聲之餘音相■者。如公與空、岡與康之類是也。取相■之字。各以類從。分為部目。謂之韻書。古今分部不同。舊詩韻共分一百六部。詩歌句尾恆用同韻之字相■。律詩隔句用韻。故兩句謂之一韻。古詩或逐句用韻。亦有數韻以後改用他韻者。謂之轉韻。
    - 風雅曰韻。如言韻人、韻事。

## 響

- 音享(Hsiang)上聲
    - 聲之感覺也。有形斯有影。有聲斯有響。故謂感應之顯見者曰影響。
