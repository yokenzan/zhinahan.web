# 首部

## 首

- 音手(Shou)上聲
    - 頭也。如古人行禮之言稽首、頓首、又民謂之黔首。言少壯黑髮之。能任國家之丁役也。
    - 君也。見（廣雅）。一國之總統、君主。皆謂之元首。
    - 長也。如謂領袖之人曰首領。第一人皆謂之首。如第一有功之人曰首功。第一富戶曰首富。是也。
    - 最先也。如謂京師曰首善之區。言開風氣之先也。
    - 篇什也。詩文一篇。或謂之一首。

- 音狩去聲
    - 有咎自呈。及告人罪曰首。如言出首、自首。

## 馘

- 音幗(Kuo)入聲
    - 截耳也。凡殺敵而獻其左耳曰馘。
