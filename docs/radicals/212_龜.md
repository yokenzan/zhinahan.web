# 龜部

## 龜

- 音規(Kuei)平聲
    - 介屬動物。腹背皆有甲。僅露頭尾四足出入之孔。甲之表面。蔽以紋片。雌者背甲隆凸。雄平。頭似蛇。有鱗。色綠。性遲純。耐饑渴。壽至百歲外。古用龜以卜。故稱靈物。以為介蟲之長。
    - 俗用為詈人之詞。唐時樂戶皆著綠頭巾。後因龜之頭為綠色。因目著綠頭巾者為龜。樂戶妻女皆歌伎。故又曰開設妓院、縱妻女賣淫者為龜。

- 音鳩(Chiu)又音邱(Ch'iu)並平聲
    - 龜茲。西域國名。卽今新疆庫車縣。

- 音麇(Chun)平聲
    - 不龜手。謂手凍不皸瘃也。見（莊子）。

- ［解字］本作龜。俗作龜。
