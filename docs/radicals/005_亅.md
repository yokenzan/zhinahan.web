# 亅部

## 亅

- 讀若橜入聲
    - 鉤之逆者也。俗謂之倒鬚鉤。

## 了

- 里曉切(Liao)上聲
    - 曉解也。如言了然於心之類。
    - 訖也。如言了事。
    - 助詞。俗語之末多用之。

## 予

- 舍呂切(Yu)上聲
    - 與也。錫與。賜與、古皆作予。
    - 我也。與余字音義並同。讀平聲。

## 事

- 仕異切(Shih)去聲
    - 人類動作之公名。（書洪範）「敬用五事。一曰貌。二曰言。三曰視。四曰聽。五曰思。」是也。其他動作。凡有意識而可名言者。皆謂之事。合種種動作。為一意識所組成者。亦謂之事。
    - 為也。如言無所事事。謂無可為之事也。
    - 侍奉也。如子事父母之類。
    - 變故也。如言四方無事。謂無禍亂之事也。
    - 物之一件。俗亦謂之一事。
