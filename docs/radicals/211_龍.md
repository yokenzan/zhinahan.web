# 龍部

## 龍

- 閭容切(Lung)平聲
    - 舊說謂為鱗蟲之長。能興雲雨。利萬物。故為四靈之一。又以喻君上。故天子卽位曰龍飛。
    - 馬高八尺以上為龍。
    - 山脈曰龍。如相地者之言龍脈。

- 與寵同。榮寵、亦曰龍光。

## 龐

- 蒲厖切(P'ang)平聲
    - 雜亂貌。又姓也。通作龎。

- 音籠(Lung)平聲
    - 充實貌。面容豐滿。亦謂之臉龐。

## 龔

- 音恭(Kung)平聲
    - 與恭同。又姓也。漢之龔遂、黃霸皆循吏。故稱循吏曰龔黃。

## 龕

- 音堪(K'an)平聲
    - 塔下室也。俗因謂供佛之小室曰佛龕。
